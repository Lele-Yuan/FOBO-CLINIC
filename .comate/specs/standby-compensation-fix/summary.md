### Task 1 — `_setAMRest` 移除 `上半` 跳过保护（第 314-323 行）

```diff
- if (!next.manual && !['休', '下', '上半', '休(补)'].includes(next.type)) {
+ if (!next.manual && !['休', '下', '休(补)'].includes(next.type)) {
```

**效果**：`休(补)` 现在可以覆盖 `上半`。

| 项目 | 之前 | 之后 |

**效果**：单天白班备班也能生成次日 `休(补)`，不再因「块未完成」而遗漏补偿。
